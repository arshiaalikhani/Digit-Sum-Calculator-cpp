# 🔢 Digit Separator & Sum Calculator (C++)

A simple and clean C++ program that receives an integer from the user, separates its digits from left to right, and calculates the sum of all digits.

This project was created to practice basic C++ programming concepts such as functions, loops, integer division, and the modulus operator.

---

## ✨ Features

- ✅ Counts the number of digits
- ✅ Separates each digit from left to right
- ✅ Calculates the sum of all digits
- ✅ Uses multiple functions for better code organization
- ✅ Handles the special case of input `0`

---

## 🛠️ Technologies Used

- C++
- iostream Library

---

## 📚 Concepts Practiced

- Functions
- While Loop
- For Loop
- Integer Division (`/`)
- Modulus Operator (`%`)
- Variables and Data Types
- User Input and Output

---

# 🚀 How It Works

The program works in three main steps:

1. Counts how many digits the number has.
2. Finds the highest place value of the number.
3. Extracts each digit and adds them together.

---

# 💻 Example

### Input:


58372


### Output:


Digits from left to right:

5
8
3
7
2

The sum of the digits is: 25
